edgeList 字段说明（边缘时间列表、以边缘为判断依据为主导）
 1.edgeList是个数组 描述鱼到达屏幕边缘的时间
 
 2.e 为0的时候表示 当前鱼在 t秒 即将游出屏幕 
{"e":0,"t":11.08}
 
 3.e 为 1的时候表示 当前鱼在 t秒 即将游入屏幕
{"e":1,"t":13.51}

 使用：
 判断生效时:数组等于空 或者 数据长度等于0 或 出生时间大于 e==1的t 值

 特殊状况：
   如果鱼一直在屏幕内游，edgeList 则为空 或者 长度为0
   如果鱼从屏幕内出生游出屏幕外则 数据 只存在  例：[{"e":0,"t":13.51}] ，游出屏幕的描述数据
   这个时 该数据的意义：鱼出生时间大于13秒时无法锁定
